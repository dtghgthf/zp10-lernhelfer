## DIE APP – VOLLSTÄNDIGE SPEZIFIKATION

### Stack
- **Eine einzige .html-Datei** – kein Build-Tool, kein npm, kein Server
- Vanilla HTML + CSS + JavaScript
- localStorage für Datenpersistenz
- OpenRouter API (fetch, kein SDK) für KI-Funktionen
- Modell: `anthropic/claude-3.5-sonnet`
- Fonts: Fontshare CDN (Cabinet Grotesk + Satoshi)
- Design: Nexus Design System (CSS Custom Properties, Dark/Light Mode)

### Navigation
4 Tabs (Desktop: Sidebar; Mobile: Bottom-Tab-Bar):
1. 🏠 Dashboard
2. 📅 Kalender
3. 📓 Tagebuch
4. ⚙️ Einstellungen

## OPENROUTER API-INTEGRATION

### Basis-Call
```javascript
async function callOpenRouter(prompt, apiKey) {
  const res = await fetch('https://openrouter.ai/api/v1/chat/completions', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${apiKey}`,
      'Content-Type': 'application/json',
      'HTTP-Referer': 'https://zp10-lernhelfer.local',
      'X-Title': 'ZP10 Lernhelfer'
    },
    body: JSON.stringify({
      model: 'anthropic/claude-3.5-sonnet',
      messages: [{ role: 'user', content: prompt }],
      response_format: { type: 'json_object' }
    })
  });
  const data = await res.json();
  return JSON.parse(data.choices[0].message.content);
}
```
